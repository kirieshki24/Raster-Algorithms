# Raster Algorithms
 
 ## Запуск Python файла:
 1. Скачать и распаковать zip-архив с исходным кодом
 2. Установить Python 3.11
 3. В командной строке(Windows) ввести следующие команды:
    ```
    chdir ...\Raster-Algorithms (Вместо троеточия вставьте путь до директории с .py файлом)
    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt
    python lab1.py
    ```
